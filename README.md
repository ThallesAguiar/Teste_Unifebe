# Teste_Unifebe
Teste para a vaga de programador PHP no Centro Universitário de Brusque - UNIFEBE


DESCRIÇÃO DO PROBLEMA:

TESTE PROGRAMADOR WEB JÚNIOR



O Núcleo de Informática da UNIFEBE, recebeu a seguinte demanda: 


Para a “Minha Nova Unifebe” se faz necessário cadastrar pessoas, sendo elas do tipo Aluno, Professor e Colaborador. Cada pessoa possui um nome, endereço completo, idade, telefone e um cadastro de um contato (apenas nome) para casos emergenciais.
Sabe-se que a linguagem Web padrão utilizada no desenvolvimento da “Minha Nova Unifebe” é PHP e o seu banco de dados é o MySQL, construa em PHP native, uma tela para cadastro e demonstre como ficaria a estrutura de Banco de Dados (DER), conforme solicitado na demanda. 


Apresente também: 


1- Rotina para emissão de relatórios de pessoa por cidade, onde o resultado retorna com as colunas Nome, Endereço (com cidade) e Telefone;



2- Rotina de consultas, com filtros por nome de pessoa, contato emergencial, cidade e idade;


3- O sistema deverá permitir, incluir, alterar ou excluir uma pessoa;
